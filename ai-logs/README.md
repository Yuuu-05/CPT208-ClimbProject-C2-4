# AI Logs

This folder documents representative AI-assisted prompts used during the development of the HoldLight web prototype and process portfolio.

These logs are included for coursework transparency. They should show how AI tools supported coding, layout refinement, accessibility checks, debugging, and portfolio writing, while the project motivation, user research findings, design logic, and evaluation interpretation remained the group’s own work.

Each file should eventually include:
- The date or approximate project stage
- The AI tool used
- The prompt or summarized prompt
- The intended task
- How the output was checked or modified by the team
- Any accessibility, ethics, or reliability considerations

Current files are placeholders and should be replaced with real prompts before final submission.
